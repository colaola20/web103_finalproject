# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

[The Student,The Creative,User]

## Draft User Stories

As a Student, I want to create a note for my "Web Dev" class so that I can keep track of project deadlines.
As a Student, I want to add tags like "Urgent" or "Review" to my notes so that I can prioritize my study sessions.
As a Student, I want to edit the text of an existing note so that I can update my information when things change.
As a Student, I want to use the AI Smart-Fix tool on my lecture notes so that I can instantly fix typos made while typing quickly during class.

As User, I want my notes to be saved in a database so that I can access them from any device without losing my data.
As User, I want to delete notes that are no longer relevant so that my board stays clean and organized.
As User, I want to search my notes by specific keywords or tags so that I can find information from weeks ago in seconds.

As a Creative, I want to change the color of my brainstorming notes to bright yellow so that I can visually distinguish "ideas" from "finalized tasks."
As a Creative, I want to use the Tone Shifter to turn my casual rough thoughts into a professional pitch so that I am ready to share my ideas with others.
As a Creative, I want to toggle between a "Grid View" and a "List View" for my notes so that I can choose the best layout for my current creative flow.
